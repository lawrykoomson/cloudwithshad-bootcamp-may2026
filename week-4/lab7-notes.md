# cloudwithlarry — Lab 7: Smart Doc Analyzer

## What I Built
- S3 bucket with uploads/ and results/ folders
- DynamoDB table for storing analysis results
- Lambda function (Python 3.12) with Textract + NLP
- S3 trigger on uploads/ prefix
- Auto-analyzes any uploaded image/PDF document
- Extracts text, sentiment, entities and key phrases

## AWS Resources Created
- S3 Bucket: cloudwithlarry-doc-analyzer
- DynamoDB Table: cloudwithlarry-doc-results
- IAM Role: cloudwithlarry-doc-analyzer-role
- Lambda Function: cloudwithlarry-doc-analyzer
- Runtime: Python 3.12 | Memory: 512MB | Timeout: 60s

## How It Works
1. Upload image/PDF to S3 uploads/ folder
2. S3 triggers Lambda automatically
3. Textract extracts all text from the document
4. Python NLP analyzes sentiment and entities
5. Results saved to DynamoDB + S3 results/ folder
